# FTNN
A small project for self backtest.

只是一個小項目用來測試自己短線購買股票的方法是否有效。 (Using FTNN API)
如果你覺得這個程式有用， 對你有幫助， 又或者希望增加什麼功能， 請告訴我。
我會願意為一個有人使用的項目花費時間去寫文檔和更新。

祝你們成功~

目前程式預設連接FUTT官方測試伺服器， 實際使用建議使用自己的伺服器。
